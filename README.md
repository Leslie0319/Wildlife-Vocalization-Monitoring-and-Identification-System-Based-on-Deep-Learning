# Wildlife-Vocalization-Monitoring-and-Identification-System-Based-on-Deep-Learning
Construction and Research of a Wildlife Vocalization Monitoring and Identification System Based on Deep Learning

This research aims to develop a wildlife vocal signature monitoring and identification system based on deep learning. Through advanced audio recognition algorithms, it can achieve automated analysis of field audio data, thereby accurately identifying and monitoring specific wildlife species. It can break the traditional ecological investigation model that relies on human intervention, providing an efficient and non-intrusive key technical means for biodiversity conservation and endangered species monitoring. It has significant ecological value and application prospects.

Work Content:
(1) Construction of wildlife sound signature dataset.
(2) Research on audio event detection and recognition methods.
(3) System development: Develop a web system with functions such as sound management, audio upload, and display of recognition results.
